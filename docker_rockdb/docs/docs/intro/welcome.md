# Welcome to RockDB

RockDB is a modern Oracle Observability and Management tool designed to provide real-time insights, persistent monitoring, and easy management of your Oracle database fleet.

## Project Overview

This documentation site is your central reference for everything related to RockDB, including its architecture, monitoring stack, and configuration.

### Core Features

- **Real-time Monitoring**: Integrated Oracle Observability Exporter.
- **Persistent Metrics**: Time-series data stored in InfluxDB.
- **Visual Dashboards**: Pre-configured Grafana dashboards.
- **Time Machine**: Historical session and SQL tracking.
- **Centralized Management**: Manage multiple databases from a single interface.
